Module Dependencies

gen_pass → def_pass subclasses; inherits all directive handlers
def_pass → expression · do_control · lexer · symbol_table · value
gen_pass → object_writer · listing_writer emits bytes & records lines
expression → lexer · symbol_table · value
symbol_table → value stores Value per SymbolEntry & ControlSection
do_control → lexer reads Statement.command to find DO/FIN/ELSE/PEND
